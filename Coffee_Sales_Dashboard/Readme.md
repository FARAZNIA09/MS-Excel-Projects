# Coffee Shop Sales Dashboard (Excel)

## Business Problem
The coffee shop wanted to analyze monthly revenue trends and top-selling products.

## Dataset
- Order date
- Product category
- Revenue
- Quantity

## Analysis Performed
- Pivot tables
- Revenue trends
- Category performance
- KPI metrics

## Key Insights
- Weekend sales were 32% higher
- Cold beverages contributed 45% revenue
